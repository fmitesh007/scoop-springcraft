# scoop-springcraft

Scoop bucket for [springcraft](https://github.com/fmitesh007/springcraft-cli)

## Installation

```powershell
# Add the bucket
scoop bucket add springcraft https://github.com/fmitesh007/scoop-springcraft

# Install springcraft
scoop install springcraft
```

## Usage

```powershell
springcraft .
```

## Uninstall

```powershell
scoop uninstall springcraft
scoop bucket rm springcraft
```
